# Southern Water Corp Project Overview

This project investigates Southern Water Corp pump performance data to understand patterns related to pump failure. It combines spreadsheet data and Python analysis to compare raw pump measurements with rolling standard deviation metrics, identify abnormal behavior, and explore indicators associated with failure events.

## Main Files

- `Southern Water Corp Economics Case Study.xlsx`: Source workbook containing the case study data.
- `Southern_Water_Corp_Python.ipynb`: Python notebook for descriptive statistics, visual analysis, filtering failure and non-failure records, quartile calculations, and outlier review.

## Key Topics

- Pump failure classification using binary failure indicators
- Descriptive statistics for raw and rolling standard deviation datasets
- Boxplot and line plot analysis of pump behavior
- Comparison of normal operation versus failed pump states
- Outlier detection using quartiles and interquartile range
- Exploration of variables such as volumetric flow, pump speed, pump torque, ambient temperature, horsepower, and pump efficiency
