# SQL Project Overview

This project is a SQL case study for the American Energy Market Regulator (AEMR). It analyzes approved energy outage records from 2016 and 2017 to evaluate market stability, outage frequency, outage duration, provider reliability, and energy loss risk.

## Main File

- `AEMR_SQL_Case_Study.ipynb`: Jupyter Notebook containing SQLite queries, business questions, and analysis for energy outage events.

## Key Topics

- Approved outage event analysis for 2016 and 2017
- Outage reason comparison across forced, consequential, scheduled, and opportunistic outages
- Monthly outage frequency trends
- Average outage duration by participant and outage type
- Risk classification based on outage duration, outage count, and forced outage criteria
- Energy stability and market reliability assessment
