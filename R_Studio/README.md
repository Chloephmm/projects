# R Studio Project Overview

This project analyzes United Nations voting behavior using R Markdown. It processes historical UN voting records, summarizes voting trends over time and by country, and uses visualization and regression methods to examine patterns in international voting behavior.

## Main File

- `UN_Voting.Rmd`: R Markdown report that loads and cleans UN voting data, calculates the percentage of yes votes, summarizes results by year and country, visualizes country-level trends, and performs linear regression analysis.

## Key Topics

- UN voting record processing
- Country code conversion and year derivation
- Percentage of yes votes across all records
- Voting trends by year and country
- Country comparisons including the United States, United Kingdom, France, and India
- Data visualization with `ggplot2`
- Linear regression for voting trend analysis
