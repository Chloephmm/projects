# AirBnB Project Overview

This project analyzes New York Airbnb listing data using a Jupyter Notebook. The notebook focuses on exploring listing distribution, neighborhood performance, room type revenue patterns, and basic revenue estimates derived from listing price and number of reviews.

## Main File

- `Airbnb_NY.ipynb`: Python notebook that imports the Airbnb dataset, explores the data with Pandas, creates calculated revenue fields, groups listings by neighborhood and room type, and visualizes revenue trends with Matplotlib.

## Key Topics

- Airbnb listing counts by New York neighborhood group
- Percentage share of listings across Manhattan, Brooklyn, Queens, Bronx, and Staten Island
- Revenue calculation using price and number of reviews
- Average revenue comparison by neighborhood group
- Top revenue-generating neighborhoods in Manhattan, Brooklyn, and Queens
- Room type revenue patterns within selected neighborhoods
