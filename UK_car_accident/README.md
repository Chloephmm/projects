# UK Car Accident Project Overview

This project cleans and prepares UK car accident data for analysis, with a focus on London road accident conditions. The notebook examines weather, road surface, lighting, and accident severity fields, then removes inconsistent or unused data before exporting a cleaned dataset.

## Main File

- `Data Cleansing.ipynb`: Python notebook that imports the accident dataset, reviews key fields, removes unused columns, corrects text values, filters invalid records, handles missing values, converts dates, and exports cleaned data.

## Key Topics

- Data cleansing for road accident records
- Weather condition review
- Road surface condition review
- Light condition review
- Accident severity cleanup
- Missing value handling
- Date conversion and cleaned data export
